# nadi-commit

**Generate a commit message from your staged git changes.**

A small everyday tool that demonstrates `priority: "cost"`. Writing a commit message from a diff is something almost any model can do well, so there's no reason to burn money on a top-tier model. The Conductor will route this to one of the cheapest models you have configured, and the result is usually indistinguishable from what GPT-4 would write.
