# nadi-shell

**Translate natural language requests into shell commands. Auto-detects whether you're on PowerShell or bash.**

A daily-driver utility that demonstrates how cheap routing works for well-defined translation tasks. Most language models are excellent at "describe what you want, get a shell command," so there's no need for a top-tier model. The Conductor routes this to something efficient, the response comes back in under a second, and you barely spend any money.
