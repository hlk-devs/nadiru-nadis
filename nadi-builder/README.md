# nadi-builder

Build Nadis with AI. Describe what you want, get reviewed code.

## How it works

1. Describe your Nadi in plain English
2. A quality AI model writes the code
3. A DIFFERENT AI model independently reviews it
4. You get the final, reviewed script

The writer and reviewer are always from different AI providers, ensuring independent code review with different training perspectives.

## Usage

1. Make sure the Nadiru engine is running on localhost:8765
2. Open index.html in your browser

## Requirements

- Nadiru engine running on localhost:8765
- At least 2 paid providers configured (for independent write + review)
